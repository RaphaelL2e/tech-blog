# [](#只出现一次的数字)[只出现一次的数字](https://leetcode-cn.com/problems/single-number/)

### [](#描述)描述

给定一个非空整数数组，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。

#### [](#说明)说明

你的算法应该具有线性时间复杂度。 你可以不使用额外空间来实现吗？

### [](#示例)示例

#### [](#示例-1)示例 1:

 输入: [2,2,1]
输出: 1 

#### [](#示例-2)示例 2:

输入: [4,1,2,1,2]
输出: 4

class Solution {
    public int singleNumber(int[] nums) {
        int ans = 0;
        Set<Integer> set = new HashSet<>();
        for(int i=0;i<nums.length;i++) {
        	if(set.contains(nums[i])) {
        		set.remove(nums[i]);
        	}else {
        		set.add(nums[i]);
        	}
        }
        ans = set.iterator().next();
        return ans;
    }
}

### [](#笔记)笔记

此处用了hash表来判断是否存在两次，最后剩下的那个则是唯一的元素
