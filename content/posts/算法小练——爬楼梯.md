# [](#爬楼梯)[爬楼梯](https://leetcode-cn.com/problems/climbing-stairs/)

### [](#描述)描述

假设你正在爬楼梯。需要 n 阶你才能到达楼顶。

每次你可以爬 1 或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？

#### [](#注意：给定-n-是一个正整数。)注意：给定 n 是一个正整数。

class Solution {
    public int climbStairs(int n) {
        int ans=0;
        if(n == 1){
            ans=1;
            return ans;
        }
        int[] dp = new int[n+1];
        dp[1] = 1;
        dp[2] = 2;
        for (int i = 3; i <=n ; i++) {
            dp[i] = dp[i-1] +dp[i-2];
        }
        ans =dp[n];
        return ans;
    }
}
