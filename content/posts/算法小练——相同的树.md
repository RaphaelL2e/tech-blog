# [](#相同的树)[相同的树](https://leetcode-cn.com/problems/same-tree/)

### [](#描述)描述

给定两个二叉树，编写一个函数来检验它们是否相同。

如果两个树在结构上相同，并且节点具有相同的值，则认为它们是相同的。

### [](#示例)示例

#### [](#示例-1)示例 1:

输入:       1         1
                / \       / 
              2   3     2   3
[1,2,3],   [1,2,3]

输出: true

#### [](#示例-2)示例 2:

输入:      1          1
               /              
            2                  2
[1,2],     [1,null,2]

输出: false

#### [](#示例-3)示例 3:

输入:       1         1
                / \       / 
              2   1     1   2
[1,2,1],   [1,1,2]

输出: false

/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode(int x) { val = x; }
 * }
 */
class Solution {
    public boolean isSameTree(TreeNode p, TreeNode q) {
        if(p==null && q==null){
            return true;
        }
        if(p==null || q==null){
            return false;
        }
        if(p.val == q.val){
            if(isSameTree(p.left,q.left)){
                return isSameTree(p.right,q.right);
            }else {
                return false;
            }
        }else {
            return false;
        }
    }
}
