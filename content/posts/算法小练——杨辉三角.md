# [](#杨辉三角)[杨辉三角](https://leetcode-cn.com/problems/pascals-triangle/)

### [](#描述)描述

给定一个非负整数 *numRows，*生成杨辉三角的前 *numRows* 行。

*

在杨辉三角中，每个数是它左上方和右上方的数的和。

输出:

    [1,1],
   [1,2,1],
  [1,3,3,1],
 [1,4,6,4,1]

class Solution {
    public List<List<Integer>> generate(int numRows) {
        List<List<Integer>> lists = new ArrayList<>();
        if(numRows==1){
            lists.add(Arrays.asList(1));
            return  lists;
        }
        if(numRows==2){
            lists = generate(1);
            lists.add(Arrays.asList(1,1));
            return lists;
        }
        if(numRows>2){
            int realNumRows = numRows;
            lists = generate(--numRows);
            List<Integer>  alist= lists.get(--numRows);
            List<Integer> blist = new ArrayList<>();
            blist.add(1);
            for (int i = 1; i < realNumRows-1; i++) {
                blist.add(alist.get(i-1)+alist.get(i));
            }
            blist.add(1);
            lists.add(blist);
            return lists;
        }else {
            return lists;
        }
    }
}
