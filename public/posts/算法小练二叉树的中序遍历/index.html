二叉树的中序遍历

给定一个二叉树，返回它的中序 遍历。

示例

1 2 3 4 5 6 7 8 输入: [1,null,2,3] 1
2 / 3

输出: [1,3,2]

代码

  • Definition for a binary tree node.

  • public class TreeNode {

  • int val;

  • TreeNode left;

  • TreeNode right;

  • TreeNode(int x) { val = x; }

  • } */ class Solution { List list = new ArrayList<>(); public List inorderTraversal(TreeNode root) { if(root!=null){ inorderTraversal(root.left); list.add(root.val); inorderTraversal(root.right); } return list; } }
